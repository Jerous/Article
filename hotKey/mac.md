# 基本快捷键
**⌘** 是Mac里最重要的修饰键，在大多数情况下相当于Windows下的Ctrl。

```
 ⌘ + Z 撤销　
 ⌘ + X 剪切　　
 ⌘ + C 拷贝（Copy）　　
 ⌘ + V 粘贴　　
 ⌘ + A 全选（All）　　
 ⌘ + S 保存（Save)　　
 ⌘ + F 查找（Find）
```

## 截图
```
⌘ + ⇧ + 4       截取所选屏幕区域到一个文件　　
⌘ + ⇧ + 3       截取全部屏幕到文件　　
⌘ + ⇧ + ⌃ + 3   截取全部屏幕到剪贴板　　
⌘ + ⇧ + 4       截取所选屏幕区域到一个文件，或按空格键仅捕捉一个窗口　　
⌘ + ⇧ + ⌃ + 4   截取所选屏幕区域到剪贴板，或按空格键仅捕捉一个窗
```

## 在应用程序中
```
⌘ + H       隐藏（Hide）当前正在运行的应用程序窗口　　
⌘ + ⌥ + H   隐藏（Hide）其他应用程序窗口　　
⌘ + Q       退出（Quit）最前面的应用程序　　
⌘ + ⇧ + Z   重做，也就是撤销的逆向操作　　
⌘ + ⇥       在打开的应用程序列表中转到下一个最近使用的应用程序，相当于Windows中（Alt+Tab）　　
⌘ + ⌥ + esc 打开“强制退出”窗口，如果有应用程序无响应，可在窗口列表中选择强制退出
```
## 文本处理
```
⌘ +    右箭头 将光标移至当前行的行尾　　
⌘ +    B 切换所选文字粗体（Bold）显示　　
fn + ⌫ 相当于PC全尺寸键盘上的⌫，也就是向后删除　　
fn + ↑ 向上滚动一页（Page Up）　　
fn + ↓ 向下滚动一页（Page Down）　　
fn + ← 滚动至文稿开头（Home）　　
fn + → 滚动至文稿末尾（End）　　
⌘ + →  将光标移至当前行的行尾　　
⌘ + ←  将光标移至当前行的行首　　
⌘ + ↓  将光标移至文稿末尾　　
⌘ + ↑  将光标移至文稿开头　　
⌥ + →  将光标移至下一个单词的末尾　　
⌥ + ←  将光标移至上一个单词的开头　　
⌃ + A  移至行或段落的开头
```

## Finder
```
⌘ + ⌥ + V 剪切文件　　
⌘ + ⇧ + N 新建文件夹（New）　　
⌘ + ⇧ + G 调出窗口，可输入绝对路径直达文件夹（Go）　　
⏎         这个其实不算快捷键，点击文件，按下可重命名文件　　
⌘ + O     打开所选项。在Mac里打开文件不像Windows里直接按Enter　　
⌘ + ⌥ + V 作用相当于Windows里的文件剪切。在其它位置上对文件复制（⌘ + C），在目的位置按下这个快捷键，文件将被剪切到此位置　　
⌘ + ↑/↓   文件夹前进，后退
⌘ + ⌫     将文件移至废纸篓　　
⌘ + ⇧ + ⌫ 清倒废纸篓　　
Space     快速查看选中的文件，也就是预览功能
```

## 在浏览器中
```
⌃ + ⇥     转向下一个标签页
⌃ + ⇧ + ⇥ 转向上一个标签页
⌘ + ⌥ + → 转向下一个标签页
⌘ + ⌥ + ← 转向上一个标签页
⌘ + [num] 转到第 num 个标签　　　
⌘ + L     光标直接跳至地址栏　　
⌘ + +/=   放大页面　　
⌘ +  +      缩小页面
```

## Mac启动与关机时的快捷键
```
⌘ + ⌥ + P + R 重置NVRAM　
　
⌥ 开机后立即按下，将显示启动管理器，如果Mac装有双系统或者插有启动U盘，可在启动管理器中选择启动盘　　

⌘ + R 开机后立即按下，可打开OS X的恢复功能（Recovery）
　　
⌘ + ⌥ + P + R 开机后立即按下，重置NVRAM。有些时候电脑会出现些小问题，重置NVRAM是你除了重新启动，尝试修复的第一选择。
　　
⌘ + ⌥ + ⌃ + 电源按钮 退出所有应用程序，允许你进行文稿储存，然后关机　
　
按住电源按钮5秒 强制Mac关机
```
